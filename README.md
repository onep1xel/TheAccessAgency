# Hugo

Hugo [Installation](https://gohugo.io/installation/)

## Run server
`hugo server`

## Development
**Install**  
`npm install`

**To start developing:**  
`npm run start`

**To generate the site HTML:**  
`npm run build`

**npm run dev** will run two commands parallel:  
`npx tailwindcss -i ./assets/css/main.css -o ./assets/css/style.css --watch`

and
`hugo server`
